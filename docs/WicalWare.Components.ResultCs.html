Namespace WicalWare.Components.ResultCs

Classes

Option<TSome>

A wrapper that helps prevent passing null values within code.

It can come in 2 different flavors.

  • None, which indicates that there was no value contained.
  • Some(TSome), which contains a wrapped value of type TSome. This value can be extracted from the Option using the Unwrap functions.

PanicException

Represents an erroneous condition encountered in Optionss and Results.

It is intended to reflect the panic semantics in Rust.

Result<TOk, TErr>

A wrapper that helps prevent passing null values within code.

It can come in 2 different flavors.

  • Ok(TOk), which contains a wrapped value of type TSome. This value can be extracted from the Option using the Unwrap functions.
  • Err(TErr), which contains a wrapped error of the type TErr. This value can be extracted from the Result using the UnwrapErr functions.

Enums

OptionKind

An enum that helps identify if the Option has a value or not.

ResultKind

An enum that helps identify if the Result contains a value or an error.
